# faniefikri.com

Personal website and project hub. Built with plain HTML, CSS, and a lot of AI assistance.

## About

This is my personal home on the internet. It lists my projects, background, and ways to connect. Built as a learning project while teaching myself web development with AI.

## Tech Stack

- HTML5 + CSS3 (no frameworks)
- Google Fonts
- Hosted on Cloudflare Pages
- Domain: faniefikri.com

## Project Structure

```
faniefikri.com/
├── index.html
├── style.css
├── assets/
│   ├── og-image.png
│   └── (other assets)
├── README.md
└── .gitignore
```

## Development

Clone the repo and open `index.html` in your browser. That's it. No build step, no dependencies, no package manager.

```bash
git clone https://github.com/faniefikri/faniefikri.com.git
cd faniefikri.com
open index.html
```
